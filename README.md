# jobs.israelis.nl

Landing page and resource hub for Israeli job seekers relocating to the Netherlands. Hebrew (RTL), AI resume tools, blog articles, and coaching services.

**Live:** [jobs.israelis.nl](https://jobs.israelis.nl)
**Staging:** [jobs-staging-x4ws.onrender.com](https://jobs-staging-x4ws.onrender.com/)

## Quick Start

```bash
npm install
npm run dev        # http://localhost:3000
```
